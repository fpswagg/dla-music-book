# Agent Planning Documents

These documents are created during planning and read before implementing each phase.
Each file corresponds to a project phase and contains the specification for what to build.

Before implementing any phase:
1. Read `docs/CONTEXT.md` for project conventions
2. Read `docs/STYLE.md` for design system rules
3. Read the corresponding phase document from this folder
